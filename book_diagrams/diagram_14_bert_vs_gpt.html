BERT: Masked Language Modeling (MLM)
Input: "The [MASK] sat on the mat"
Target: predict "cat" from full context
Loss: −log P(cat | The, _, sat, on, the, mat)

Also: Next Sentence Prediction (NSP) — is B the next of A?
GPT: Causal Language Modeling (CLM)
Input: "The cat sat on the"
Target: predict "mat" from left context only
Loss: −Σ log P(xₜ | x₁…xₜ₋₁)

Same objective as generation — training = inference!

Encoder-Decoder (T5, BART)
Hybrid: BERT encoder reads full input, GPT-style decoder generates output autoregressively. Best for conditional generation: translation, summarization, question answering with long context.
BERT's bidirectional attention produces richer token representations — "bank" in "river bank" can attend to "river" even if it appears later. GPT's causal mask ensures that training loss at every position uses only its valid left-context, making generation and training mathematically identical. This is why GPT-style models dominate modern LLMs: one training objective (CLM) directly optimizes the generation capability.
Python Demo — stdlib only
import math

def softmax(v):
    m=max(v); e=[math.exp(x-m) for x in v]; t=sum(e)
    return [x/t for x in e]

def make_bert_mask(n):
    "Full attention — no masking"
    return [[1]*n for _ in range(n)]

def make_gpt_mask(n):
    "Causal mask — lower triangular"
    return [[1 if j<=i else 0 for j in range(n)] for i in range(n)]

def masked_attention_weights(n, mask_fn):
    "Compute attention weights with a given mask."
    raw = [[1.0 - 0.1*abs(i-j) for j in range(n)] for i in range(n)]
    mask = mask_fn(n)
    weights = []
    for i in range(n):
        row = [raw[i][j] if mask[i][j] else -1e9 for j in range(n)]
        weights.append(softmax(row))
    return weights

tokens = ["The", "cat", "sat", "on"]
n = len(tokens)

# How many positions each token can attend to
for name, mask_fn in [("BERT", make_bert_mask), ("GPT", make_gpt_mask)]:
    weights = masked_attention_weights(n, mask_fn)
    print(f"\n{name} attention weights:")
    print(f"{'':6s}" + " ".join(f"{t:6s}" for t in tokens))
    for i, (tok, row) in enumerate(zip(tokens, weights)):
        vals = " ".join(f"{v:.3f}" if v > 0.001 else "  --- " for v in row)
        print(f"Q={tok:4s}  {vals}")
