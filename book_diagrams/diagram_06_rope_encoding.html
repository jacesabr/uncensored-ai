Rotary Positional Encoding (RoPE)
Used in LLaMA, GPT-NeoX, PaLM — encodes position as rotation angle in 2D subspaces
Query q at position m — rotated by m·θ
·
Key k at position n — rotated by n·θ
Key Property
⟨R(m·θ)q, R(n·θ)k⟩
= ⟨q, R((n-m)·θ)k⟩
The dot product depends only on the relative distance (n−m) between positions, not their absolute values.
Rotation applied per 2D pair of dimensions

Applied to Q/K directly (not added to input), so relative distance is naturally encoded in the dot product — not just the vectors.
Adoption
LLaMA 1/2/3, Mistral, Falcon, GPT-NeoX, PaLM 2, Gemma. Dominant positional encoding in modern open-source LLMs.
RoPE encodes position m by rotating each 2D pair of dimensions of Q and K by angle m·θᵢ. The inner product of a rotated Q at position m with a rotated K at position n depends only on their relative offset (m−n), giving the model natural sensitivity to distance without absolute position leakage.
Python Demo — stdlib only
import math

def rope_rotate(x, pos, base=10000):
    """Apply RoPE rotation to vector x at sequence position pos."""
    d = len(x)
    out = list(x)
    for i in range(0, d, 2):      # process pairs (x2i, x2i+1)
        theta = pos / (base ** (i / d))
        cos_t, sin_t = math.cos(theta), math.sin(theta)
        x0, x1 = x[i], x[i+1]
        out[i]   = x0 * cos_t - x1 * sin_t
        out[i+1] = x0 * sin_t + x1 * cos_t
    return out

def dot(a, b): return sum(x*y for x,y in zip(a,b))

# Show that dot product depends only on relative distance
d = 8  # try 4 or 16
q = [1.0 if i%2==0 else 0.5 for i in range(d)]
k = [0.8 if i%2==0 else 0.3 for i in range(d)]

print("Relative distance → dot product (should depend only on gap):")
for gap in [0, 1, 2, 5, 10]:
    scores = []
    for m in range(5):     # different absolute positions
        n = m + gap
        rq = rope_rotate(q, m)
        rk = rope_rotate(k, n)
        scores.append(round(dot(rq, rk), 4))
    variance = max(scores) - min(scores)
    print(f"  gap={gap:2d}: scores={scores}  variance={variance:.4f}")
