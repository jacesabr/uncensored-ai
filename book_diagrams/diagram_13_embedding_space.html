Vector arithmetic — analogies emerge from geometry
vec("king") − vec("man") + vec("woman") ≈ vec("queen")
vec("Paris") − vec("France") + vec("Germany") ≈ vec("Berlin")
vec("walked") − vec("walk") + vec("run") ≈ vec("ran")
These vector offsets encode relational structure: gender, capital-country, verb tense. The same offset "works" across many pairs.

Relational offsets
The vector from "man" to "woman" approximates the same offset as "king" to "queen". This linear structure emerges from PMI matrix factorization implicitly.
gender_dim ≈ vec("woman") − vec("man")
Degenerate subspaces
Embeddings can encode harmful biases (nurse→female, doctor→male from training data). Debiasing methods project out these directions.
StereoSet, WinoBias: measures embedding bias
Word embeddings (Word2Vec, GloVe, or the token embeddings of LLMs) map each token to a dense vector in R^d. Geometric proximity correlates with semantic similarity. PCA or t-SNE projections reveal clusters: animals, royalty, geography, etc. The key insight from Word2Vec (Mikolov et al., 2013): the geometry of the embedding space encodes relational knowledge through vector arithmetic.
Python Demo — stdlib only
import math, random

def cosine(a, b):
    dot = sum(x*y for x,y in zip(a,b))
    na = math.sqrt(sum(x**2 for x in a))
    nb = math.sqrt(sum(x**2 for x in b))
    return dot / (na * nb + 1e-9)

def vec_add(a,b): return [x+y for x,y in zip(a,b)]
def vec_sub(a,b): return [x-y for x,y in zip(a,b)]

# Toy 4D embeddings with deliberate structure
# dims: [royalty, gender, geography, temperature]
embeddings = {
    "king":    [0.9,  0.8,  0.0,  0.0],
    "queen":   [0.9, -0.8,  0.0,  0.0],
    "man":     [0.1,  0.9,  0.0,  0.0],
    "woman":   [0.1, -0.9,  0.0,  0.0],
    "Paris":   [0.0,  0.0,  0.9,  0.0],
    "France":  [0.0,  0.0,  0.7,  0.0],
    "hot":     [0.0,  0.0,  0.0,  0.9],
    "cold":    [0.0,  0.0,  0.0, -0.9],
    "warm":    [0.0,  0.0,  0.0,  0.5],
}

# Test analogy: king - man + woman ≈ queen
analogy = vec_add(vec_sub(embeddings["king"], embeddings["man"]), embeddings["woman"])
print("Analogy: king - man + woman")
results = [(cosine(analogy, v), w) for w, v in embeddings.items() if w != "king"]
for sim, word in sorted(results, reverse=True)[:4]:
    bar = "█" * int(max(0,sim)*20)
    print(f"  {word:8s} {bar} {sim:.3f}")
