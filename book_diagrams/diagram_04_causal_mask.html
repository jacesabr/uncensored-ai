Key property
Row i can only attend to columns 0…i.
Token at position i cannot see future tokens.
How the mask is applied
mask[i,j] = 0 if j > i (future → masked with −∞)
mask[i,j] = 1 if j ≤ i (past/present → kept)

# After softmax, −∞ → 0.0 (zero attention to future)
score_masked = score + (1 - mask) × (-1e9)
attn_weights = softmax(score_masked / √d_k)
Training
All positions trained in parallel — each position's loss uses only its allowed context. Teacher forcing with causal mask enables O(seq) positions simultaneously.
Inference
At generation time step t, only position t generates new logits. KV cache stores past positions — no recomputation needed.
BERT difference
BERT uses no causal mask — all positions attend to all positions bidirectionally. This enables better understanding but cannot generate autoregressively.
The causal mask is a lower-triangular boolean matrix. When added to raw attention logits as −∞, the softmax operation converts those positions to exactly 0.0. During training, this allows computing loss for all sequence positions simultaneously — a critical efficiency gain over RNNs.
Python Demo — stdlib only
import math

def softmax(v):
    m = max(v)
    e = [math.exp(x - m) if x > -1e8 else 0.0 for x in v]
    t = sum(e) or 1e-9
    return [x/t for x in e]

def causal_mask(n):
    "Lower-triangular mask: 1 where allowed, 0 where masked"
    return [[1 if j <= i else 0 for j in range(n)] for i in range(n)]

def apply_mask_and_softmax(scores, mask):
    masked = [
        [s + (0 if m else -1e9) for s, m in zip(row, mrow)]
        for row, mrow in zip(scores, mask)
    ]
    return [softmax(row) for row in masked]

tokens = ["The", "cat", "sat", "on"]
n = len(tokens)

# Fake attention scores (Q·Kᵀ / √d_k)
scores = [[0.5 * (n - abs(i-j)) for j in range(n)] for i in range(n)]
mask = causal_mask(n)
weights = apply_mask_and_softmax(scores, mask)

print(f"{'':8s}" + " ".join(f"{t:6s}" for t in tokens))
for i, (tok, row) in enumerate(zip(tokens, weights)):
    vals = " ".join(f"{v:.3f}" if v > 0.001 else "  --- " for v in row)
    print(f"Q={tok:5s}  {vals}")
